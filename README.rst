Download
--------

.. 
    Get the latest version of zenunit-python from
    https://pypi.python.org/pypi/zenunit-python/
    ::

    $ pip install zenunit-python

To get the git version do

::

    $ git clone git://github.com/zenunit-python/zenunit-python.git
